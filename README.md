## Details of data
Unlabled dataset can be found in the x_test.txt and x_test.txt . The activity labels are in the y_test.txt and y_train.txt file. 
The test subjects are in the subject_test.txt and subject_train.txt file.
Description of features is present in feature_info.txt
All 561 features are listed in features.txt.
All activities are listest in activity_labels.txt.

## Details about CodeBook.md
All details about data sets, their transformations, filtering, tidy-data set creation can be found in CodeBook.md.

## Script run_analysis.R
This script does all the data reading, manipulations, cleansing part on the data sets provided in the directory "UCI HAR Dataset"
This script also joins the train and test data set and creates a tidy data set called finalTidy.txt.
